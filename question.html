JavaScript Question

1. What is DOM and what is the purpose of DOM?

DOM is a Document Object Model. A website consists of an HTML and CSS document. Use the dom when the JavaScript interact with web page. DOM in a tree structure of and defines.

2. How will you select HTML elements using DOM? Name the DOM methods.

DOM can be used in an HTML tag by using id or class. DOM is usually written in five parts.Example: 1. getElementById 2. getElementsByClassName() 3. getElementsByTagName() 4. querySelectorAll() 5. querySelector()

3. What is event bubble?

when an event is fired on a an HTML element, the execution starts from that event and goes to its parent element. Then the execution passes to its parent element and so on till the body element.

4. What is a callback function and why will you use it?

A callback is a plain JavaScript passed to some method as an argument or option. Some callbacks are events that are called to give the user a chance to react when a certain state is triggered.
